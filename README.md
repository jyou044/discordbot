# discordbot
Simple discord bot created through use of Python
## Overview
### Commands
- **$hello**: Sends "Bonjour"
- **$yogi**: Sends "I love donuts!"
- **$oops**: Sends "D'OH!"
- **$photo**: Sends randomized photo
- **$meme**: Sends randomized GIF
